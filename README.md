## 3. Descripción general del sistema

El presente proyecto consiste en el desarrollo de un sistema de gestión de supermercado implementado en lenguaje Python y ejecutado mediante consola. Su objetivo es simular el funcionamiento básico de una caja registradora, permitiendo visualizar un catálogo de productos, agregar artículos al carrito de compras, modificar o eliminar productos seleccionados, calcular automáticamente subtotales, descuentos y promociones, generar un ticket de compra y registrar estadísticas generales de ventas. El sistema fue desarrollado utilizando estructuras de datos básicas, funciones, estructuras de control y manejo de excepciones, aplicando los contenidos trabajados durante la asignatura.

## 4. Instrucciones de ejecución

Para ejecutar el sistema, seguir los siguientes pasos:

1. Descargar o clonar el repositorio del proyecto desde GitHub.
2. Abrir la carpeta del proyecto utilizando Visual Studio Code o cualquier otro editor compatible con Python.
3. Abrir una terminal dentro de la carpeta donde se encuentra el archivo `gestion_supermercados.py`.
4. Verificar que Python se encuentre instalado en el equipo.
5. Ejecutar el programa mediante el siguiente comando:

```bash
python gestion_supermercados.py
```

En algunos sistemas también puede utilizarse:

```bash
python3 gestion_supermercados.py
```

6. Una vez iniciado el programa, se mostrará un menú por consola con las distintas opciones disponibles.
7. Seleccionar la opción deseada ingresando el número correspondiente y seguir las instrucciones indicadas en pantalla para utilizar el sistema.
